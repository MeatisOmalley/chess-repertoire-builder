Chess Repertoire Builder Privacy Policy

Chess Repertoire Builder accesses chessboard content on supported Chess.com and Lichess pages, including the current position, moves, and board context, solely to provide repertoire-building, study, and live-practice features.

The extension stores repertoires, imported PGNs, chapter labels, annotations, practice progress, preferences, and—if the user connects Lichess—a Lichess OAuth token locally in the browser.

During Live Practice, the extension sends the current chess position in FEN format and the selected rating to Lichess Opening Explorer to obtain aggregated move statistics. When the user explicitly connects Lichess, the OAuth token is sent only to Lichess for that service.

The developer does not operate a data-collection server and does not sell, rent, share, profile, advertise with, or use this data for any purpose unrelated to the extension’s chess features. Users can remove local data by deleting repertoires, disconnecting Lichess, clearing the extension’s data, or uninstalling the extension.
